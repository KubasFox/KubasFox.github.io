O projektu

Grid Fotografií

Tento projekt představuje moderní responzivní galerii fotografií.

Funkce

  • Responzivní design pro všechny zařízení
  • Interaktivní grid s fotkami
  • Elegantní animace a přechody
  • Možnost zvětšení fotografií
  • Moderní uživatelské rozhraní

Technologie

Projekt využívá čistý HTML, CSS a JavaScript bez externích knihoven, což zajišťuje rychlé načítání a kompatibilitu.
